# Quiz Mania

A Dynamic Quiz Application Where you can Challenge Yourself with a Diverse Range of Topics and Put Your Expertise to the Test!

# Technologies that I use:
 1. ReactJs
 2. React Router Dom
 3. Context API
 4. React Fontawesome
 5. React Toastify
 6. Lottie React
 7. React Rechart
 8. Tailwind CSS
